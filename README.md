### Installation

Game requires [Node.js + npm](https://nodejs.org/) v6.9+ and [webpack](https://webpack.github.io/) v1.13+ to run.

```sh
$ git clone https://github.com/sns2009/newaudioplayer.git
$ npm install
$ npm run production
$ npm run start
```
